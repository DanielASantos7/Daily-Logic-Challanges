# Count Duplicates (6 kyu)

**Link:** https://www.codewars.com/kata/54bf85e3d5b56c7a05000cf9

## Descrição

Contar quantos caracteres distintos aparecem mais de uma vez em uma string, ignorando diferenças entre maiúsculas e minúsculas.

## Aprendizados

- Uso de `Dictionary` para contagem de frequência.
- Manipulação de strings com `ToLower`.
- Iteração e filtragem de dados.
