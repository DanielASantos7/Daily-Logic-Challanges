# Convert to Camel Case (6 kyu)

**Link:** https://www.codewars.com/kata/517abf86da9663f1d2000003

## Descrição

Converter uma string separada por hífens ou underscores em camelCase. A primeira palavra mantém o formato original, enquanto as demais devem iniciar com letra maiúscula.

## Aprendizados

- Manipulação de strings em C#.
- Uso de `Split` com múltiplos separadores.
- Manipulação de caracteres com `char.ToUpper`.
- Uso de `Substring` para reconstrução de palavras.
