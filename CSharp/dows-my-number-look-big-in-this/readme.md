# Narcissistic Number (6 kyu)

**Link:** https://www.codewars.com/kata/5287e858c6b5a9678200083c

## Descrição

Verificar se um número é narcisista, ou seja, se a soma de seus dígitos elevados à potência da quantidade de dígitos é igual ao próprio número.

## Aprendizados

- Conversão de número para string.
- Iteração com `foreach`.
- Uso de `Math.Pow`.
- Manipulação de caracteres numéricos.
