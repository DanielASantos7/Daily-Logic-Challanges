# Row Sum Odd Numbers

## Descrição

Dado um triângulo formado por números ímpares consecutivos, retornar a soma dos números da linha n.

## Exemplo

1 → 1  
2 → 3 + 5 = 8  
3 → 7 + 9 + 11 = 27

## Solução

A soma da linha n segue o padrão matemático n³, permitindo uma solução direta sem necessidade de loops.

## Complexidade

O(1)
