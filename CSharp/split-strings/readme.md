# Split Strings (6 kyu)

**Link:** https://www.codewars.com/kata/515de9ae9dcfc28eb6000001

## Descrição

Dividir uma string em pares de dois caracteres. Caso a string tenha tamanho ímpar, o último par deve ser completado com "\_".

## Aprendizados

- Manipulação de strings em C#.
- Uso de loops com incremento personalizado (de 2 em 2).
- Verificação de limites para evitar erros de índice.
