# Regex Validate PIN Code (7 kyu)

**Link:** https://www.codewars.com/kata/55f8a9c06c018a0d6e000132/solutions/csharp

## Descrição

O objetivo é validar se um PIN contém exatamente 4 ou 6 dígitos numéricos.

## Aprendizados

- Manipulação de strings em C#.
- Uso de `char.IsDigit` para validação de tipos.
- Refatoração para evitar códigos aninhados (Guard Clauses).
